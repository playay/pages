###vimrc

```
"这个文件中, 等号两边不能有空格

"语法高亮
syntax on

"状态栏标尺
set ruler

"显示行号
set number

"高亮当前行
set cursorline

"自动缩进
set autoindent

"换行符
set fileformat=unix

"制表符转换成空格
set expandtab

"制表符宽度
set tabstop=4

"格式化时制表符的宽度
set shiftwidth=4

"连续的空格转化成制表符（如：移动光标时，一次四个空格）
set softtabstop=4 

"启用插件:
```
